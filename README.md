# AI Bulk Trainer

Aplicación web con IA para generar planes personalizados de entrenamiento y nutrición.

Ideal para personas que quieran introducirse en el mundo del gimnasio, pero no tienen mucha idea y no quieren gastar dinero.

Con esta aplicación puedes introducir tus datos, nombre, edad, peso etc... y apartir de tu información generamos con IA un plan "personalizado" a tu necesidad.

Tecnologias utilizadas:
- HTML
- CSS
- JavaScript
- Node.js
- Express
- OpenAI
